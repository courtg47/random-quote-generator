A Pen created at CodePen.io. You can find this one at http://codepen.io/courtg47/pen/aBqyaw.

 This random quote machine will give you an inspirational quote by clicking the "New Quote" button.  It will generate a new quote at random with each click.  You may also tweet the quote with the "Tweet this" button which will take you to a new page and automatically insert the quote into the tweet function. This project was created by Courtney Green.
